# 🧘 Stress Relief Hub — Complete Mental Wellness App

A comprehensive Streamlit application for stress management, anxiety relief, and mental wellness.

## Features

### 🚨 Immediate Relief
- **Panic SOS Button** — Instant grounding exercises & breathing pacers
- **Physiological Sighs** — Immediate stress reduction technique
- **5-4-3-2-1 Grounding** — Sensory awareness exercise

### 🧠 Overthinking & Anxiety Tools
- **Worry Time Scheduler** — CBT-based delayed worrying
- **Cognitive Reframing Journal** — Identify & rewrite cognitive distortions
- **Brain Dump / Thought Jar** — Voice & text racing-thought capture

### 🌙 Sleep & Sound
- **Sleep Stories** — Narrated bedtime stories with ambient sound
- **Soundscapes** — Rain, ocean, white noise, brown noise, pink noise
- **Binaural Beats** — Alpha, theta, delta wave generation
- **Sleep Timer** — Auto-stop after set duration

### 🫁 Breathwork Library
- **4-7-8 Breathing** — For sleep and anxiety
- **Box Breathing** — For focus and performance
- **Coherent Breathing** — 5 breaths per minute
- **Physiological Sigh** — Double inhale, long exhale
- **Custom Pacer** — Visual breathing guide

### 📊 Tracking & Insights
- **Daily Mood Check-ins** — Quick emotion logging
- **Mood Graphs** — Pattern visualization over time
- **Trigger Analysis** — Identify stress patterns
- **Streak Tracking** — Consistency rewards

### 🤖 AI & Personalization
- **AI Virtual Therapist** — CBT-trained chatbot companion
- **Personalized Calm Plans** — Daily curated wellness schedules
- **Smart Recommendations** — Based on mood & time of day

### 🌱 Gamification
- **Virtual Zen Garden** — Grows with your wellness habits
- **Streak Badges** — Achievement system
- **Mindfulness Minutes** — Track total meditation time

### 👥 Community
- **Anonymous Peer Support** — Safe, moderated sharing space
- **Support Stickers** — Send encouraging reactions
- **Crisis Resources** — Immediate help hotlines

## Installation

```bash
pip install -r requirements.txt
```

## Running the App

```bash
streamlit run app.py
```

## Audio Features
All sounds are procedurally generated using numpy/scipy — no external audio files needed!
- White/Brown/Pink noise
- Binaural beats (Alpha/Theta/Delta)
- Nature-inspired tones
- Isochronic tones

## Data Storage
All user data is stored locally in SQLite (`data/wellness.db`) — fully private.
